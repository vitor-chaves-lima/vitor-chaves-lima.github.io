Wonder


Meu projeto de conclusão do curso de eletroeletrônica foi um controlador
MIDI chamado Wonder. O módulo desenvolvido precisava se comunicar via USB
com um computador, que estaria executando algum software de produção músical
com suporte para a tecnologia MIDI.


O módulo é responsável por reconhecer qual tecla do teclado foi pressionada e
enviar o código correspondente para o computador. Que iria reconhecer esse
código e emitir o som correspondente


Nesse projeto foi utilizado um microcontrolador AVR.
